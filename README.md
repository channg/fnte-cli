<h3>fnte-cli</h3>
中文文件名称转换英文cli工具😄😄

[![NPM](https://nodei.co/npm/fnte-cli.png)](https://nodei.co/npm/fnte-cli/)

<h4>Installation</h4>
<p>Prerequisites: Node.js <p>

    ```
    npm install -g vue-cli

    ```


<h4>Usage</h4>

    ```
    fnte
    ```
<h4>Do what</h4>
<p>将文件夹下的所有中文文件名转换为拼音文件名</p>
